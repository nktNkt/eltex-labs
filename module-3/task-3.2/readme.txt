Программа принимает на вход 2 аргумента: приоритеты сообщений для отправителя и получателя.

Если их не вводить, то установятся значения приоритетов 100 и 200.

Приоритет 255 используется для проверки наличия сообщений в очереди,
поэтому если какой-то аргумент равен 255, то программа завершается с ошибкой.
